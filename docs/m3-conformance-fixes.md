# M3 Conformance Fixes — resolution of `docs/m3-conformance-review.md`

> Companion to [`m3-conformance-review.md`](m3-conformance-review.md). Records how each of the **235 reviewed discrepancies** was resolved.
>
> **Method:** 28 component-scoped fixer agents applied every in-scope finding (each restricted to its own `libs/ui/<name>/src/` + doc); shared-file changes were deferred to the lead and applied serially; the lead then fixed the regressions the parallel pass introduced and verified the whole repo.
>
> **Verification (2026-06-04):** `nx run-many -t lint test build -p ui web` → **all green** (ui 329 specs, web 32 specs, both libraries build, 0 lint errors).

## Headline

- **212 discrepancies fixed in-component** by the parallel pass (measurements, color roles, states, variants, anatomy, shape, a11y, doc artifacts).
- **7 of 10 deferred shared-file changes resolved by the lead** (badge offsets, bottom-sheet margins, icon-button doc, radio keyboard-focus). **1 rejected** with evidence (global state-layer opacity). **2 left as documented residuals** + the switch ripple.
- **13 findings deliberately not changed** — M3 *usage* guidance or out-of-component-scope (e.g. carousel "Show all", linear-progress screen inset, verifier "conforms" confirmations).
- **3 regressions introduced by the parallel pass were caught by verification and fixed** (tooltip double-match, menu-item input alias, date-picker tooltip imports).

### Decision: state-layer focus/pressed opacity stays 0.12 (rejected the 0.10 change)

Two auditors disagreed: the icon-button auditor wanted the global token changed `0.12 → 0.10`; the fab finding (cross-verified) said the impl's 0.12 is correct and the doc's "10%" is the artifact. Verified against Google's own **material-web** M3 system tokens (`tokens/.../_md-sys-state.scss`): **hover 0.08, focus 0.12, pressed 0.12, dragged 0.16** — the repo's `_state.css` already matches exactly. So the global token was left unchanged and the icon-button doc was corrected to 12% (matching the already-corrected `fab.md`).
